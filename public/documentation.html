Brief intro about the the app

This project comprieses of an API that gives users access to a movie database where they can
search for movies by names, and information about the specific movies such as, director, genre, artwork,
etc. Users can also sign up, update their personal information as well as create a list of their favorite movies.

Please visit the table below to see all detailed methods and more!

Endpoints and Descriptions.

Business Logic/Request URL Endpoints Method Query Parameters Request body data format Response body data format
Return a list of all the movies to the user /movies GET None None A JSON object holding all the data about all the movies
Return data about a single movie by title to the user /movies/[title] GET :title None A JSON object holding all the data about a single movie. Example:
{
Title: "Man of Steel",
Director: "Zack Snyder",
Genre: '"Fantasy/Superhero",
}
