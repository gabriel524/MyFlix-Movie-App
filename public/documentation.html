App Documentation

Here you can find information on how to utilize the API

This project comprieses of an API that gives users access to a movie database where they can
search for movies by names, and information about the specific movies such as, director, genre, artwork,
etc. Users can also sign up, update their personal information as well as create a list of their favorite movies.
