Brief intro about the the app

This project comprieses of an API that gives users access to a movie database where they can
search for movies by names, and information about the specific movies such as, director, genre, artwork,
etc. Users can also sign up, update their personal information as well as create a list of their favorite movies.

If you have any questions or suggestion, please feel free to drop me a line.

Endpoints and Descriptions.

Business Logic/Request URL Endpoints Method Query Parameters Request body data format Response body data format
Return a list of all the movies to the user /movies GET None None A JSON object holding all the data about all the movies
Return data about a single movie by title to the user /movies/[title] GET :title None A JSON object holding all the data about a single movie. Example:
{
Title: "Man of Steel",
Director: "Zack Snyder",
Genre: '"Fantasy/Superhero",
}
Return data about a genre by name movies/genre/[genre] GET :genre None A JSON object format containing movies belonging to a specific genre. Example:

{
Title: 'The Lord of the Rings: The Fellowship of the Ring',
Director: "Sir Peter Jackson",
Genre: 'Fantasy',
}
Return data about a director by name /movies/director/[director] GET :director None A JSON format data holding all the data about a director by name. Example:
{
Name:"Christopher Nolan",
Bio: "A short bio",
Dob:"July 30, 1970"

}
Allow users to add a movie to their list of favorites /favourite/add/[id]/[title] POST :add A JSON format data containing all the new data about the movie to be added. Example:

{
Title: 'Man of Steel',
Director: "Zack Snyder",
Genre: 'Fantasy/Superhero',

Allow new users to register /users/register POST :added/register Data containing new user details A text response indicating that a new user was added or register
Allow users to update their user info /users/update/[id] PUT :id Data containing changes to old profile A text response indicating that the changes wer deployed successful vai update
Allow users to remove a movie from their list of favorites /favourite/delete/[id]/[title] DELETE :id, :title None An array without the removed movie
Allow existing users to deregister /users/deregister/delete/[id] DELETE :id None A text response indicating that the user profile was successfully deregistered/deleted
